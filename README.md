Este repositório contém a página de vendas do curso, hospedada gratuitamente via GitHub Pages.

￼
🚀 Como Publicar no GitHub Pages
1. Crie o Repositório
 • Acesse github.com/new
 • Nome: metodo-empregativo (ou o nome que preferir)
 • Deixe público
 • Clique em "Create repository"
2. Faça Upload dos Arquivos
￼
3. Ative o GitHub Pages
 • Vá em Settings → Pages (no menu lateral)
 • Em Source, selecione Deploy from a branch
 • Escolha a branch main e pasta /(root)
 • Clique em Save
4. Acesse Sua Página
 • Aguarde 2-5 minutos
 • Sua página estará em: https://SEU_USUARIO.github.io/metodo-empregativo/

￼
🔗 Configure o Link da Kiwify
Antes de publicar, substitua o texto SEU_LINK_KIWIFY_AQUI no arquivo index.html pelo link real do checkout da Kiwify.
O texto aparece 2 vezes no arquivo:
1. Botão principal "QUERO ACESSAR AGORA"
2. Botão sticky no final da página
